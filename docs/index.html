📦 Quick Installation

  1. Download or clone the repository: git clone https://github.com/pangerlkr/VPN-Extension.git
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable "Developer mode" in the top right corner
  4. Click "Load unpacked" and select the src directory
  5. Configure your gateway server in the extension popup
  6. Click "Connect" to start using the proxy!
