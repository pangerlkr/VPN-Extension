Frequently Asked Questions

How do I install VeilRoute Proxy?

Installation is simple! Clone the repository, open Chrome's extension page (chrome://extensions/), enable Developer mode, and load the 'src' folder as an unpacked extension. Full instructions are in the README.

Is VeilRoute Proxy free to use?

Yes! VeilRoute Proxy is completely free and open-source. However, you'll need to configure your own proxy server or gateway. The extension is just the client interface.

Does it work with SOCKS5?

Absolutely! VeilRoute supports both HTTPS and SOCKS5 proxy protocols. You can configure your preferred protocol in the extension settings.

What about WebRTC leaks?

We've got you covered! VeilRoute includes built-in WebRTC leak protection to prevent your real IP address from being exposed through WebRTC connections.

Can I contribute to the project?

We love contributions! Check out our CONTRIBUTING.md file for guidelines. Whether it's bug fixes, new features, or documentation improvements, all PRs are welcome!
